# Knowledge Synthesis Guide

The Synthesis Engine aggregates multi-source findings to expose systemic conclusions.

---

## 1. Synthesis Capabilities

1. **Common Patterns**: Exposes overlapping findings across multiple ingested papers.
2. **Contradictions Detection**: Automatically flags conflicting technical statements.
3. **Opportunities**: Maps cited areas to list concrete development projects.

---

## 2. CLI Management

```bash
# Run synthesis across project papers
aios research synthesize
```
