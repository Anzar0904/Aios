## Pipeline Stages

1. **Query Parser**: Extracts semantic intent from user query
2. **Embedding Generator**: Converts query to vector representation
3. **Vector Store**: Qdrant vector database for similarity search
4. **Embedding Cache**: LRU cache for frequently accessed embeddings
5. **Similarity Search**: K-nearest neighbors retrieval
6. **Result Ranker**: Reranks results by relevance and recency
7. **Context Builder**: Assembles final context from retrieved memories
