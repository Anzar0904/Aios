# Architecture Decision Records (ADRs)

> This section archives all Architecture Decision Records (ADRs) — the rationale, context, and consequences behind every major technical choice made in the Personal AI OS.

---

## What is an ADR?

An Architecture Decision Record captures:
- **Context**: What problem or constraint triggered the decision
- **Decision**: What choice was made
- **Rationale**: Why this option was chosen over alternatives
- **Consequences**: What trade-offs were accepted
