# Local Development Setup

> Step-by-step guide for setting up a local development environment.

## Prerequisites

- Python 3.10 or higher
- Docker and Docker Compose
- Git
- 8 GB RAM minimum (16 GB recommended)
- 20 GB disk space minimum

## Installation Steps

### 1. Clone Repository

```bash
git clone https://github.com/your-org/aios.git
cd aios
```

### 2. Create Virtual Environment & Install Dependencies

```bash
python3 -m venv .venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate  # Windows
pip install -e ./core
```

### 3. Start External Services

Start required external services using Docker Compose:

```bash
docker-compose up -d postgres redis qdrant
```

**Required services:**

| Service | Default Port | Purpose |
|---------|-------------|----------|
| PostgreSQL | 5432 | SQL persistence — workspaces, tasks, AI memory |
| Redis | 6379 | Cache, sessions, distributed locks, queues, rate limiting |
| Qdrant | 6333 | Vector store — semantic memory, conversation embeddings |

**Optional services:**

- **n8n** (port 5678) — workflow automation (`docker-compose up -d n8n`)

### 4. Configure Environment

Create a `.env` file with required configuration:

```bash
cp .env.example .env
# Edit .env with your API keys and service addresses
```

Minimum required variables for local development:

```bash
# PostgreSQL
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_DATABASE=aios
POSTGRES_USER=aios_user
POSTGRES_PASSWORD=<your-password>

# Redis
REDIS_HOST=localhost
REDIS_PORT=6379

# Qdrant
QDRANT_HOST=127.0.0.1
QDRANT_PORT=6333

# OmniRoute (at least one provider required)
OPENROUTER_API_KEY=sk-or-...
```

See [configuration.md](configuration.md) for all configuration parameters.

### 5. Configure AIOS

Review and edit `config/config.toml`:

```toml
[runtime]
name = "Personal AI OS"
version = "0.5.0"
debug = false

[llm]
provider = "openrouter"
default_model = "qwen/qwen3-coder"

[llm.openrouter]
base_url = "https://openrouter.ai/api/v1"
timeout = 30
```

### 6. Run Database Migrations

Migrations are run automatically inside `bootstrap_kernel()` on startup. To run them manually:

```bash
python -m aios.migrations
```

### 7. Start AIOS

```bash
aios
```

## Verification

Verify your setup is working:

```bash
# 1. Check AIOS version
aios --version

# 2. Test PostgreSQL connectivity
psql -U $POSTGRES_USER -d $POSTGRES_DATABASE -c 'SELECT version();'

# 3. Test Redis connectivity
redis-cli -h $REDIS_HOST -p $REDIS_PORT ping  # Expects: PONG

# 4. Test Qdrant connectivity
curl http://$QDRANT_HOST:$QDRANT_PORT/health
```
