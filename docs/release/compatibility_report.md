# Platform Compatibility Report

This document reports cross-platform compatibility and supported Python environment details.

---

## 1. Supported Environments
- **macOS**: Fully supported (tested on macOS 14/15 Darwin).
- **Linux**: Fully supported (tested on Ubuntu 22.04/24.04 LTS).
- **Windows**: Supported under WSL2 (Windows Subsystem for Linux).

---

## 2. Python Environment & Dependency Requirements
- **Python Version**: `>=3.12` and `>=3.14` compliant.
- **Dependencies**: Verified compatible with minimal footprint (zero bloat, uses standard libraries where possible).
