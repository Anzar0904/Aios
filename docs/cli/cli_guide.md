# AI OS CLI Command Guide

This guide describes all commands and configurations of the AI OS CLI.

---

## Command Categories

### 1. General & System
- `aios help`: Display general help options.
- `aios setup`: Run interactive configuration wizard onboarding.
- `aios dashboard`: Display system status metrics.
- `aios diagnostics`: View OS telemetry diagnostic details.
- `aios session`: Check current CLI session details.

### 2. Subsystems
- `aios project`: Manage project intelligence registries.
- `aios business`: Manage client agency portfolios.
- `aios approval`: Manage governance policies and approvals queue.
