# Interactive Shell Guide

This guide describes how to run and manage the interactive terminal shell.

---

## 1. Starting the Shell
To launch the interactive shell:
```bash
aios
```
On boot, the system executes startup health checks and loads your previous session state.

---

## 2. Interactive Features
- **Multiline Mode**: Switch multiline mode on/off using `/multiline`.
- **Slash Commands**: Query subsystems directly (e.g., `/project`, `/status`, `/models`).
- **Command History**: Navigate previous inputs using the Up/Down arrow keys.
- **Auto-completion**: Hit Tab to complete commands.
- **Graceful Exit**: Type `/exit` or hit Ctrl+D to save and exit.
