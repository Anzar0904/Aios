# Keyboard Shortcuts Reference Guide

This document describes all key combinations supported in the AI OS interactive shell.

---

## 1. Supported Key Combinations
- **Ctrl+C**: Cancel current input or process execution.
- **Ctrl+L**: Clear the terminal screen.
- **Ctrl+R**: Launch standard readline command history search.
- **Ctrl+K**: Open search command palette fuzzy search.
- **Tab**: Auto-complete active words.
- **Arrow keys**: Navigate history logs and edit inputs.
- **Esc**: Exit autocomplete or palette.

---

## 2. Command Palette Integration
Pressing **Ctrl+K** prints the `Search Command Palette` prompt. Type any keyword (e.g. `business`, `vercel`, `project`) to list matching commands instantly.
