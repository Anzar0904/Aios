# Changelog Engine Guide

The Changelog Engine builds historical version updates logs and release notes.

---

## 1. Capabilities

- **Changelog Generator**: Summarizes version release tags.
- **Release Notes Generator**: Generates features checklist, breaking changes alerts, and migration instructions.

---

## 2. CLI Invocation

```bash
# Generate project changelog
aios docs changelog

# Generate release notes
aios docs release
```
