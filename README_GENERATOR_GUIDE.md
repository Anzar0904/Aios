# Auto README Generator Guide

The README Generator builds a standardized README layout for any project.

---

## 1. Document Structure

The generated README includes:
1. **Header**: Name of the workspace project.
2. **Project Overview**: Description of the project purpose.
3. **Features**: List of core features.
4. **Installation**: Codeblocks outlining standard installation commands.
5. **Usage**: Overview of CLI command routes.

---

## 2. CLI Invocation

To generate a README for a workspace module:

```bash
# Generate README for project 'agency'
aios docs readme agency
```
